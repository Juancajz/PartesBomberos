# Sistema de Gestión para Bomberos

Proyecto de desarrollo de software orientado a la gestión interna de una institución de bomberos.

## Descripción
Aplicación web diseñada para apoyar la administración y operación interna de una compañía de bomberos. El sistema permite la gestión de usuarios, creación de partes, control de inventario y autenticación mediante login, centralizando la información en una plataforma web.

## Tecnologías utilizadas
- Backend: Python, Django
- Frontend: React, HTML, CSS, JavaScript
- Base de datos: PostgreSQL
- Otras librerías y herramientas asociadas al desarrollo web

## Funcionalidades principales
- Autenticación y control de acceso (login de usuarios)
- Gestión de usuarios y roles
- Creación y gestión de partes
- Gestión de inventario
- Estructura modular orientada a escalabilidad

## Estado del proyecto
Proyecto en desarrollo (aprox. 90% completado), actualmente funcional y en proceso de mejora continua.

## Objetivo del proyecto
Proyecto desarrollado con fines de aprendizaje y práctica en desarrollo web full stack, aplicando buenas prácticas de programación y diseño de sistemas.
